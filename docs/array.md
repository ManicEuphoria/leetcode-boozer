## 解题技巧

### 双指针技巧
- 快慢指针：用于原地修改数组
- 对撞指针：处理有序数组的查找
- 滑动窗口：处理子数组问题

### 排序预处理
- 适用场景：需要处理元素间顺序关系
- 注意事项：排序会改变原数组，注意是否允许修改输入

### 二分查找
- 适用场景：有序数组中的查找
- 关键点：边界条件的处理
