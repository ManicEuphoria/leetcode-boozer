## 解题技巧

### 双指针技巧
- 977题使用双指针从数组两端向中间遍历，通过比较平方值大小来构建结果数组
- 这种方法避免了显式排序，时间复杂度为 O(n)

## 注意事项
1. 考虑数组为空的边界情况
2. 注意负数的平方可能很大
3. 善用双指针降低时间复杂度
